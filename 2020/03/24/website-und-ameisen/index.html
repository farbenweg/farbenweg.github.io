Website und Ameisen

March 24, 2020

Das ist mein erster Post. Heute ist ein sonniger Märztag mit einer kalten Bise. Ich lerne es, Websites mit Hugo zu erstellen.

Dazu benutze ich die drei Komponenten Hugo, Windows PowerShell ISE und Visual Studio Code.

Eine Site erstellen

md example
hugo new site ./

config.toml anpassen:

baseURL = "http://localhost:1313/"
languageCode = "de-de"
title = "Beispiel Hugo"

Ein Theme herunterladen

cd themes
git clone https://github.com/tanksuzuki/hemingway.git
cd ../

Das Theme in der Konfigurationsdatei config.toml als solches festgelegen:

baseURL = "http://localhost:1313/"
languageCode = "de-de"
title = "Beispiel Hugo"
theme = "hemingway"

Einen Inhalt erstellen

hugo new post/erster-post.md

Es entsteht im Unterverzeichnis content/post/ eine Seite, in die ich einen Eintrag schreiben kann:

---
title: "Erster Post"
date: 2020-03-24T09:51:12+01:00
draft: true
---
Das ist mein erster Post. Heute ist ein sonniger Märztag mit einer kalten Bise.
