Favicon

04.04.2020

Seit drei Tagen übe ich mich an den Icons der Webseite. Ich habe gesehen, dass es zwei Arten von Icons gibt. Die ganz kleinen nennen sich favicon.ico oder favicon.png und befinden sich oben im Fenster oder im Tab. Sie sind 16 x 16 Pixel, 32 x 32 Pixel oder 48 x 48 Pixel klein:

favicon-16x16.png

favicon-32x32.png

favicon.ico

Dann gibt es etwas grössere Icons, die sich touch-icon nennen. Sie tauchen z.B. im Browser auf, wenn eine Seite als wichtig angesehen wird. Sie sind 180 x 180 Pixel, 192 x 192 Pixel oder 256 x 256 Pixel klein:

apple-touch-icon.png

android-chrome-192x192.png

android-chrome-256x256.png

Der Code in head.html lautet dann:

<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="android-chrome-192x192" sizes="192x192" href="/android-chrome-192x192.png">
<link rel="android-chrome-256x256" sizes="256x256" href="/android-chrome-256x256.png">

<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="icon" type="image/ico" sizes="32x32" href="/favicon.ico">

<link rel="manifest" href="/site.webmanifest">
<meta name="msapplication-TileColor" content="#da532c">
<meta name="theme-color" content="#ffffff">
