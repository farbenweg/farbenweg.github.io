Neues Layout

08.04.2020

Das neue Layout umfasst

  • neue Schriftarten,
    • Merienda für die Titel,
    • Lato für die Texte,
  • einen höheren und hellgrünen Header,
  • ein grösseres Logo im Header,
  • grüne Schrift beim Überfahren der Navigation.

Der Code in /themes/lithium/layouts/partials/head.html lautet dann:

<link href='https://fonts.googleapis.com/css?family=Merienda' rel='stylesheet'>

In /themes/lithium/static/css/main.css gab es viele kleine Änderungen:
